# Security Policy

This project controls a USB audio device over HID. Bugs are more likely to cause
bad device state, loud output, or app crashes than traditional security issues,
but reports are still welcome.

## Reporting

Please open a GitHub issue for:

- Unsafe volume behavior
- Unexpected raw HID report behavior
- Crashes caused by malformed input
- Device detection problems

Avoid publishing proprietary iBasso app files or decompiled code in issues.
